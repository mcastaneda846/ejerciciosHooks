# React Hooks Practice

Proyecto de práctica desarrollado con React, TypeScript y Vite para aprender los conceptos fundamentales de React mediante ejercicios separados por componentes.

---

# Tecnologías utilizadas

- React
- TypeScript
- Vite
- React Router DOM

---

# Objetivos del proyecto

Este proyecto fue creado para practicar:

- Componentes funcionales
- Props
- useState
- useEffect
- useRef
- useReducer
- useContext
- Formularios controlados
- Renderizado dinámico
- Navegación SPA
- Arquitectura basada en componentes
- Manejo de estado
- Validaciones básicas
